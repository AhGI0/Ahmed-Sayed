## üÜò Common Issues & Fixes

### Styles not loading?
- Make sure `style.css` is in same folder
- Try Ctrl+Shift+R (clear cache)
- Check file names match exactly

### Dark mode not working?
- Try in different browser
- Check F12 console for errors
- Try clearing localStorage

### Links not working?
- Open in text editor
- Check href values are correct
- Make sure URLs start with http://

### Mobile menu not showing?
- Resize browser window smaller
- Try on actual mobile phone
- Check `script.js` file is there
